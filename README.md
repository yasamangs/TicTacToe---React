# Requirements

- Download [nodejs](https://nodejs.org/en/)

# How to Run

There are 2 ways:
- 1. Install yarn by typing this commad in cmd :

   >  npm install -g yarn 
  2. Then type
   > yarn start

- Or simply type
    > npm start

These will open http://localhost:3000 to view it in your deafult browser.
